Car Rental

The project involves creating a comprehensive system for car rentals, which consists of three main components:


  • Web Application (CarRentalComparerApp): Allows users to browse, filter, and compare rental car offers from various providers.
  • Car Rental Comparer API (ComparerAPI): Serves as an intermediary between the web application and car providers, managing communication and data exchange. It allows integration of different Providers' APIs (adapter design pattern).
  • Car Provider API (ProviderAPI): API of a car rental company, allowing users to rent and return cars.

The system has been designed with flexibility and ease of extension in mind, allowing the addition of new car providers in the future. Each provider will be able to share their resources through a dedicated API, ensuring scalability and the ability to integrate with various partners.


An additional assumption of the system is the possibility of collaboration with other project teams, who will deliver their own car provider APIs. This ensures extensive possibilities for expanding the system with new functionalities.
